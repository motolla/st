# **Quick Keybind**
```
# Copy & Paste
Ctrl + Shift + C | Copy Selected
Ctrl + Shift + V | Paste

# Zoom
Ctrl + + | Zoom In
Ctrl + - | Zoom Out
Ctrl + 0 | Reset Zoom

# Scroll
Shift + Page_Up | Scroll Up
Shift + Page_Down | Scroll Down
```

# Requirements
+ Void
```
xbps-install pkgconf pkgconf-devel libXft-devel libX11-devel libXinerama-devel
```

# Install
**Clone repo**
```
git clone https://github.com/gillgrite/st.git
```
**Build & Install**

```
./stclean
```
